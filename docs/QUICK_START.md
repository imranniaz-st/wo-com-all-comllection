- Select an existing page OR create a new one

### Step 3: Configure Display
Set your preferred:
- **Columns**: Desktop (2-5), Tablet (1-5), Mobile (1-2)
- **Filters**: Show/hide category and color filters
- **Search**: Enable/disable search box

- Choose fonts (Poppins, Roboto, Open Sans, etc.)
- Set sizes (e.g., 18px)
- Pick font weights (300-800)
- Select colors using color picker

### Step 5: Save & Done!
Click **Save Changes** and visit your collection page.

## ðŸ†˜ Quick Troubleshooting

**Products not showing?**
- Ensure WooCommerce is active
- Check that collection page is assigned
- Verify products are published in WooCommerce

**Fonts not updating?**
- Clear browser cache (Ctrl+Shift+Delete)
- Clear any caching plugins
- Hard refresh page (Ctrl+F5)

**Elementor widget missing?**
- Reload Elementor editor
- No errors - widget only appears if Elementor is active
